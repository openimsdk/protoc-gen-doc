# ChangeGroupMemberMute

| Method Name | Request Type | Response Type | Description |
| ----------- | ------------ | ------------- | ------------- |
| ChangeGroupMemberMute | [ChangeGroupMemberMuteReq](docs/group/index.mdx#ChangeGroupMemberMuteReq) | [ChangeGroupMemberMuteResp](docs/group/index.mdx#ChangeGroupMemberMuteResp) | mute or cancel mute a group member |
 

#### ChangeGroupMemberMuteReq
| Field | Type | Label | Description |
| ----- | ---- | ----- | ----------- |
| groupID | [string](docs/scalar/scalar.mdx) |  | group id you want to mute |
| userID | [string](docs/scalar/scalar.mdx) |  | user id you want to mute |
| mutedSeconds | [uint32](docs/scalar/scalar.mdx) |  | mute time (unit: seconds), 0 cancels the mute |
